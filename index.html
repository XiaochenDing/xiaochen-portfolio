关于我

Robotic 3D Printing × Machine Learning × Architectural Design

机器人3D打印 × 机器学习 × 建筑设计

I develop responsive fabrication systems that integrate computer vision and machine learning with robotic clay 3D printing—closing the loop from sensing to corrective actuation in real time.

我专注于将计算机视觉与机器学习融入陶土3D打印，实现从感知到纠偏执行的实时闭环控制。

Research Projects

研究项目

Responsive Fabrication System for Real-Time Detection and Correction in Clay 3D Printing view

TU Delft · Graduation Project (11/2024 – 06/2025)

  • Closed-loop ML control adjusting robotic arm speed for material flow and overhang integrity.
  • Dataset across multiple speeds/inclinations; ResNet Attention‑56 & DINOv2 + Grad‑CAM.
  • Integrated extruder, Raspberry Pi cameras, PC–robot comms; validated vs. open-loop.

Live-Detection Calibration for Automated G‑Code Adjustment in Clay 3D Printing view

TU Delft · CORE Group (09/2024 – 11/2024)

  • Real-time ML calibration workflow in Python with Attention‑56 parameter prediction.
  • Dynamic parameter adjustment improved print consistency and quality.

Biophilic Informed Tool for Floor Plan Layout Generation view

TU Delft · MEGA (04/2024 – 06/2024)

  • Grasshopper-based multi-objective tool using 5 biophilic criteria (daylight/air/thermal/visual/social).
  • Outputs quantifiable scores and spider charts to balance well-being and performance.

Discrete Design for Bird Hide - the Automated Assembly with UR5 view

TU Delft · Design Informatics Project (02/2024 – 04/2024)

  • Optimized toolpath, pickup method, and station design for UR5, synchronizing digital and physical models to enhance human–robot interaction.
  • enabled efficient prototype-scale robotic assembly of discrete components.

Rule‑Based Modular Design — Oral History Library (Cockatoo Island) view

University of Sydney · Undergraduate Thesis (08/2020 – 11/2020)

  • Enhanced WASP framework with secondary boundary constraints for customized discrete spaces.

Pedestrian Wind Comfort Parametric Tool view

University of Sydney · Algorithmic Architecture (08/2019 – 09/2019)

Data‑Driven Plan Generation for Residential Buildings

Realize Hyperboloid Shell Roof Design in a Low‑Cost Condition
